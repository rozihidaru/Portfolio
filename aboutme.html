The Core 4 of Coding

Conditionals: A conditional runs code if a certain requirement is met. If (something): do x. Else: do y. A real-life example would be that if you get an A on your test, your parents will let you hang out with your friends. If not, you have to stay home and study. Variables: A variable stores information. Variables can be booleans (True/False), integers (numbers), or strings (text). It's just like variables in a math class, like y=8 and x=3. Functions: A function defines code so it is easier and less complicated for the coder to use. Loops: A loop repeats a part of code as many times as indicated by the coder.
